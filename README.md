# ATM Simulator System

## 📌 Project Overview
The **ATM Simulator System** is a Java-based desktop application that simulates the basic operations of an Automated Teller Machine (ATM). It allows users to perform banking transactions such as depositing money, withdrawing cash, checking balance, and viewing transaction history.

The project is developed using **Java Swing for the graphical user interface** and **MySQL for database management**, providing a simple simulation of real-world ATM banking operations.

---

## 🚀 Features
- User Login using PIN authentication  
- Account Registration System  
- Deposit Money  
- Withdraw Cash  
- Fast Cash Option  
- Balance Enquiry  
- Mini Statement (Transaction History)  
- Change PIN  
- User-friendly GUI using Java Swing  

---

## 🛠️ Technologies Used
- **Java**
- **Java Swing (GUI)**
- **MySQL Database**
- **JDBC Connectivity**

---

## 📂 Project Structure
Some of the important files in this project include:

- `Login.java` – Handles user login  
- `Signup.java` – New account registration  
- `Deposit.java` – Deposit money into account  
- `Withdrawl.java` – Withdraw money from account  
- `BalanceEquiry.java` – Check account balance  
- `MiniStatement.java` – View transaction history  
- `FastCash.java` – Quick withdrawal option  
- `Transactions.java` – Main transaction interface  
- `Conn.java` – Database connection  

---

## ⚙️ How to Run the Project

1. Clone the repository
```
git clone https://github.com/sumanthmarcos34/ATM-Simulator-System.git
```

2. Open the project in **IntelliJ IDEA / Eclipse / NetBeans**

3. Setup **MySQL Database**

4. Update database credentials in:
```
Conn.java
```

5. Run:
```
Login.java
```
